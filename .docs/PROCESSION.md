# Procession — 进程列队，系统成诗

> **Procession**: Process + Procession（巡游/行列）
> 把冷冰冰的系统监控升格为动态图谱艺术，名字本身就是叙事。

## 一句话

Procession 是一款桌面端应用，将你的电脑运行状态实时映射为一座**活着的 3D 城市景观**——默认是桌面艺术品，交互可切换为实用监控工具。

## 项目定位

| 维度 | 定位 |
|------|------|
| 类型 | 桌面原生应用（Tauri） |
| 目标用户 | 开发者、技术爱好者、桌面美化爱好者 |
| 核心价值 | **把系统监控从枯燥的数字变成美的体验** |
| 调性 | 精致、诗意、赛博朋克但不阴暗 |
| 竞品区分 | 不是更好看的任务管理器，是**让监控成为艺术**的全新品类 |

## 视觉隐喻

```
每个进程 → 一座建筑
CPU 占用 → 建筑高度（动态呼吸）
内存占用 → 建筑底面积
进程树 → 城市街区布局（根进程在市中心，子进程向外辐射）
进程类型 → 建筑颜色（系统蓝、用户暖、活跃白金）
网络连接 → 建筑间的光缆（光点沿缆线流动）
磁盘 I/O → 地面下的能量脉动
GPU 负载 → 天空风暴强度
温度 → 建筑辉光色调（低→蓝，高→红）
```

## 核心体验

默认状态下，Procession 静静地运行在你的桌面上，像一幅活的数字画。偶尔瞟一眼，你能从城市的"天气"和"建筑群"的起伏中，直觉感知到系统的状态。

当你需要真正了解发生了什么时：

1. 鼠标悬停/点击 → 查看进程详情
2. 双击 → 镜头飞向该进程
3. 空格键 → 切换实用模式，叠加上 HUD 仪表盘和建筑标注

## 技术信条

- **后端只做事，不做视觉** — Rust 层只采集系统数据，不碰任何渲染逻辑
- **IPC 是推模式，不是拉模式** — 后端每秒推送整帧快照，前端插值渲染
- **本地优先，不依赖云** — 所有处理在本地完成
- **跨平台可移植** — PlatformAdapter 抽象，Windows 首发，macOS 紧随
